Movie Matrix API Documentation

The Movie Matrix API will be a database for the server-side of the Movie Matrix web application. It will provide information about movies, genres, and directors. Users will be able to create a profile for saving their favorite movies.

Endpoints Table

Request Description Endpoint URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get list of movies /movies GET none none An array of JSON objects holding data about all movies in database
Get list of movies by director /movies/directors/[director] GET :director none An array of JSON Objects holding data about all movies by requested director
Get list of movies by genre /movies/genres/[genre] GET :genre none An array of JSON Objects holding data about all movies in requested genre
Get data about a single movie by title; containing: title, description, director, year released, genre, image URL, featured or not /movies/[title] GET :title none A JSON object holding data about a single movie. Example:
{
movieID: 005,
title: 'Almost Famous',
description: 'A high-school boy in the early 1970s is given...'
director: 'Cameron Crowe',
release: '2000'
genre: ['Adventure', 'Comedy', 'Drama', 'Music'],
imageURL: 'almostFamous.png',
}
Get data about director by name; containing: bio, birth year, death year /directors/[director] GET :director none A JSON Object holding data about a single director. Example: {
directorName: 'Cameron Crowe',
directorBio: 'Certainly idiosyncratic as a writer, Cameron Crowe has created a series of scripts...',
birthYear: '1957',
deathYear: 'null'
}
Get data about a specific genre by name; containing: genre name, description /genres/[genre] GET :genre none A JSON Object holding data about a specific genre. Example:
{
genre: 'Comedy',
genreDes: 'A comedy film is a category of film that emphasizes humor...'
}
Allow new users to register /users POST none A JSON Object holding data about new registered user. Example:
{
registeredName: 'Marco Esquandolas',
userName: 'antelope',
userPassword: 'runrunrun',
userEmail: 'outofcontrol@gmail.com',
userBirthday: '1983-12-02'
} 		A JSON Object holding data about new registered user including userID. Example:
{
userID: '001',
registeredName: 'Marco Esquandolas',
userName: 'antelope',
userPassword: 'runrunrun',
userEmail: 'outofcontrol@gmail.com',
userBirthday: '1983-12-02',
topMovies: []
}
Allow users to update user info /users/[userName] PUT :userName none A text message indicating User info was sucessfully updated
Allow users to update favorite movies list /users/[userName]/topMovies/[movieID] PUT :userName, :movieID none A text message indicating that the User suscessfully added a movie to their Favorite Movie List.
Allow users to remove a movie from their favorite movies list /users/[userName]/topMovies/[movieID] DELETE :userName, :movieID none A text message indicating that the User sucessfully removed movie from their Favorite Movie List.
Allow existing users to remove account /users/[userName] DELETE :userName none A text message indicating the Users Account was sucessfully removed.
