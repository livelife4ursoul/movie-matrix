Movie Matrix API Documentation

The Movie Matrix API will be a database for the server-side of the Movie Matrix web application. It will provide information about movies, genres, and directors. Users will be able to create a profile for saving their favorite movies.

Endpoints Table

Request Description Endpoint URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Get list of movies /movies GET none none An array of JSON objects holding data about all movies in database
Get list of movies by director /movies/directors/:directorName GET :directorName none An array of JSON Objects holding data about all movies by requested director
Get list of movies by genre /movies/genres/:genreName GET :genreName none An array of JSON Objects holding data about all movies in requested genre
Get data about a single movie by title; containing: title, description, director, year released, genre, image URL, featured or not /movies/:Title GET :Title none A JSON object holding data about a single movie. Example:

Title: 'Almost Famous',
Description: 'A high-school boy in the early 1970s is given...'

Name: 'Comedy',
Description: 'A comedy film is a category of film that emphasizes humor...' }

Bio: 'Certainly idiosyncratic as a writer, Cameron Crowe has created a series of scripts...',
Birth: '1957',
Death: 'null' }
ImagePath: 'almostFamous.png' }
Get data about director by name; containing: bio, birth year, death year /directors/:directorName GET :directorName none A JSON Object holding data about a single director. Example:
{ Director: {
Name: 'Cameron Crowe',
Bio: 'Certainly idiosyncratic as a writer, Cameron Crowe has created a series of scripts...',
Birth: '1957',
Death: 'null' }
}
Get data about a specific genre by name; containing: genre name, description /genres/:genreName GET :genreName none A JSON Object holding data about a specific genre. Example:
{ Genre: {
Name: 'Comedy',
Description: 'A comedy film is a category of film that emphasizes humor...' }
}

Lookup user info by Username /users/:Username GET :Username none A JSON Object holding data about user
Allow users to update user info /users/:Username PUT :Username none A text message indicating User info was sucessfully updated
Allow users to update favorite movies list /users/:Username/movies/:MovieID PUT :Username, :MovieID none A text message indicating that the User suscessfully added a movie to their Favorite Movie List.
Allow users to remove a movie from their favorite movies list /users/:Username/movies/:MovieID DELETE :Username, :MovieID none A text message indicating that the User sucessfully removed movie from their Favorite Movie List.
Allow existing users to remove account /users/:Username DELETE :Username none A text message indicating the Users Account was sucessfully removed.
